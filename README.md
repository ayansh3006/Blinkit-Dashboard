# 📊 Power BI Sales Dashboard Project

## 🚀 Overview

This project is an interactive **Sales Dashboard** built using Power BI.
It provides insights into sales performance, profit trends, and customer behavior to support data-driven decision making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and regions
* Track profit trends over time
* Understand customer purchasing behavior

---

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel / CSV (Dataset)
* Data Visualization Techniques

---

## 📂 Dataset

The dataset used in this project includes:

* Sales data
* Product details
* Customer information
* Regional performance

*(Dataset source: Based on tutorial project)*
